本题套用模板

```java
    public int lengthOfLongestSubstring(String s) {
        char[] cs = s.toCharArray();
        int max = 0;
        Set<Character> set = new HashSet<>();
        for(int l = 0 , r = 0 ; r < cs.length ; r++){
            char c = cs[r];
            while(set.contains(c)){
                set.remove(cs[l]);
                l++;
            }
            set.add(c);
            max = Math.max(max,r-l+1);
        }
        return max;
    }
```
